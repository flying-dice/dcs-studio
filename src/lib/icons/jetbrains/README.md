# IntelliJ Platform icons

SVGs vendored from [JetBrains/intellij-community](https://github.com/JetBrains/intellij-community)
(`platform/icons/src/expui/fileTypes` and `expui/nodes`), browsable at
<https://intellij-icons.jetbrains.design/>. Copyright 2000-2023 JetBrains s.r.o.
and contributors, used under the Apache License 2.0.

`lua.svg` / `lua_dark.svg` and `dcsManifest.svg` / `dcsManifest_dark.svg` are
our own derivatives in the same style (not part of the upstream set). `*_dark.svg` variants are used when the chrome is dark;
`gitignore` has no dark variant upstream and falls back to the light one.
